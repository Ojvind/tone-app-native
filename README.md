# Guess the Notes

A mobile app for practicing music note reading. Built with React Native and Expo.

## About

Each round displays 8 random notes — 4 in treble clef and 4 in bass clef — and the player types in what they think each note is called. After checking the answers, correct and incorrect guesses are revealed and the score is tallied across the whole game.

The number of rounds can be configured before the game starts.

## Getting started

```bash
npm install
npx expo start
```

## Tech

- React Native (Expo)
- VexFlow for note rendering (via WebView)
- iOS and Android
